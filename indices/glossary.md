# Glossary（术语表）

> 目标：让人和 Agent 在同一套术语上对齐，避免口径漂移。

## 1️⃣ 交易与风险
- **Slippage（滑点）**：理想成交价与实际成交价的差异
- **Market Impact（冲击成本）**：你的交易行为对价格造成的影响
- **Drawdown（回撤）**：净值从峰值到谷值的回落幅度

## 2️⃣ 数据与可复现
- **SSOT（Single Source of Truth）**：单一真相来源（不要把关键结论散落在聊天里）
- **Replay（回放）**：能用同样数据与流程复现同样结果

## 3️⃣ LLM / Agents
- **Spec-driven（规约驱动）**：先写规格与验收，再实现
- **Idempotency Safety Wall（幂等安全墙）**：把确定逻辑关进脚本/工具，缩小非幂等区域
