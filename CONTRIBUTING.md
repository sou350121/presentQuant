# Contributing（给未来的自己 + Agent）

## 1. 最小提交单元

- 日更：1 篇 `daily/YYYY-MM-DD.md`
- 或主题沉淀：1 个 `taxonomy/<topic>/README.md` 的小步增量
- 或索引：`indices/*` 的一次更新

## 2. 内容结构（强制）

每篇日更必须包含：
- 来源（链接/截图）
- 你的 3 点结论
- 机制（因果链）
- 行动项（3-7 条）
- 边界与反例

## 3. 引用规范

- 能给链接就给链接
- 不要编造数据/结论
- 没证据就写“观察/推测”，并写清“需要补哪些证据”

## 4. 命名约定

- `daily/YYYY-MM-DD.md`
- `taxonomy/<topic>/README.md`
